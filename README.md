# @infusedui/core

> [!IMPORTANT]
> To add `@infusedui` to you project you need to create a `.npmrc` file at the same folder base of you `package.json` with this line
>
> ```npmrc
> @infusedui:registry=https://npm.pkg.github.com
> ```
